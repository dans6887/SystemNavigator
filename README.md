# SystemNavigator

System Navigator is an administration Suite designed to simplify the use of Linux Terminal and Windows Command Line commands. The functions are modualrized to reduce resource overhead. The main functions are listed for the user at the beginning of the program run. The main menu will reload after running the desired function or until the user terminates the program.
